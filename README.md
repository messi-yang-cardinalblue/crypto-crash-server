# Crypto Crash Server

### Start the server

```bash
yarn
yarn build
yarn start
```

## Development

### Install packages

```bash
yarn
```

### Develop

Develop with hot-reload enabled.

```bash
yarn dev
```

### Build

This will compile Typescript and generate Javascript in **/dist** folder.

```bash
yarn build
```

### Start Server

Build and start the server.

```bash
yarn start
```

### Unit Tests

Run all unit tests.

```bash
yarn test
```

Or build unit tests whiling watch it.

```bash
yarn test:watch
```

### Lint Checks

```bash
yarn lint
```

### Pre Commit Hook

Please make .husky/pre-commit executable

```bash
chmod +x .husky/pre-commit
```
